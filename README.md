## 🌟 `폼프` 구현 목표

### `일반회원`

폼프를 이용해주시는 주고객 

- 회원에게 상황에 적합한 운동 추천서비스
- 전문적인 클래스 매칭 서비스
- 회원간 운동모임 소모임 서비스

### `크리에이터`

폼프를 통해서 운동 클래스를 제공하는 멘토

- 크리에이터가 추구하는 클래스 등록
- 개설된 클래스 조회 및 관리
- 클래스에 관한 일반회원의 문의 관리

### `관리자`

폼프 서비스를 전반적으로 컨트롤

- 일반회원과 크리에이터 회원 관리
- 공지사항&FAQ 서비스
- 일반회원과 크리에이터 1:1문의 관리
